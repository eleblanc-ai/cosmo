# Slices

This directory tracks completed work slices. Cosmo creates files here as work is approved.

## Format

Each slice is recorded as: `SLICE-NNN-short-description.md`

Use 1-3 word descriptions:
- `SLICE-001-project-setup.md`
- `SLICE-002-task-list.md`
- `SLICE-003-add-form.md`

## Contents

Each slice file contains:
- Date completed
- What was implemented
- Files created/modified
- Review results (tests passed, etc.)

This creates a reviewable history of the project's development.
